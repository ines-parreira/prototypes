Order status, tracking & delivery timing
Handles customer questions about where an order is, when it will arrive, and how to track a shipment.
When to use
Trigger this skill when a customer asks about the status of a placed order, a tracking number, or the expected delivery window.
What the skill does
Looks up the customer's most recent order using their email.
Returns the current fulfilment status (processing, shipped, out for delivery, delivered).
Shares the tracking link and estimated delivery date when the order has shipped.
Escalates to a human if the order is delayed beyond the expected window.
Tone
Friendly and reassuring. Acknowledge any delay and give the customer a clear next step.
